BestFriends

BestFriends is a web app based on the MySpace feature that allows users to have their top friends. The desire to build this came from wanting to build a small app, but implement it with very useful features, like auth, email notifications and to also learn a new tech stack.

I wanted to build this app with multiple devs with the desire to be the lead dev so I could learn strong development practices. Like how to design a system and choose the right components for each part of the application.
